breakdown

nesquena / hermes-webui

This interface provides a three-panel layout that mirrors Hermes CLI features in a mobile-responsive web environment. It includes a visual context ring for real-time token monitoring and a dedicated workspace file browser for managing agent assets.

fathah / hermes-desktop

This desktop companion simplifies the initial setup of Hermes Agent through guided installation and automated dependency resolution. It serves as a centralized hub for managing 16 messaging gateways, scheduled tasks, and agent personas.

EKKOLearnAI / hermes-web-ui

A comprehensive dashboard focused on multi-platform connectivity, supporting eight messaging channels including WhatsApp and Slack. It features advanced usage analytics for tracking costs and model distribution alongside robust session management.

outsourc-e / hermes-workspace

This native web workspace introduces a 'Swarm Mode' for dispatching multiple persistent agents with specific roles. It integrates a terminal and memory browser directly into the dashboard while maintaining zero-fork compatibility with vanilla Hermes installations.

dodo-reach / hermes-desktop

A native macOS application that establishes direct SSH connections to remote Hermes hosts without requiring intermediary APIs. It organizes agent activity through a Kanban-style interface and provides local workflow presets for embedded terminal tasks.

xaspx / hermes-control-interface

A self-hosted management gateway that allows users to start, stop, and configure individual agent profiles from a single dashboard. It features a split-view file editor with syntax highlighting and real-time system metrics for performance tracking.

clawvader-tech / hermes-telegram-miniapp

This mobile-first React dashboard is designed specifically as a Telegram Mini App, supporting terminal-style chat and slash commands. It enables background agent spawning for up to five concurrent instances with real-time hardware monitoring.

pyrate-llama / hermes-ui

Built on a single-file React architecture, this glassmorphic interface offers real-time log streaming and inline workspace previews. It provides a streamlined environment for managing MCP tools, skills, and cron jobs through a modern web UI.

aivrar / portable-hermes-agent

A zero-install Windows GUI that bundles over 100 tools, including TTS and ComfyUI, without requiring Docker or admin rights. It features a dynamic tool maker for creating API wrappers at runtime and granular permission controls for local models.

sanchomuzax / hermes-webui

A monitoring-centric dashboard that provides full-text search across conversation histories and detailed inspection of message logs. It focuses on service health, cost tracking, and the management of scheduled cron jobs and skill sources.

Euraika-Labs / pan-ui

A chat-first workspace featuring real-time SSE streaming and interactive tool approval cards for human-in-the-loop workflows. It includes an integrated marketplace for MCP servers and isolated profile management for distinct memory and API keys.
