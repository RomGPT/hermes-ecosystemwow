breakdown

nesquena / hermes-webui

This lightweight Python and vanilla JavaScript interface provides full feature parity with the Hermes CLI in a mobile-friendly web format. It features a three-panel layout that allows users to browse workspace files and view inline previews alongside their chat sessions.

EKKOLearnAI / hermes-web-ui

A comprehensive dashboard focused on multi-platform connectivity, offering unified configuration for eight channels including Telegram, Discord, and WhatsApp. It includes robust session management, scheduled job tracking, and detailed token analytics for precise cost estimation.

outsourc-e / hermes-workspace

This native web workspace introduces 'Swarm Mode' for role-based dispatching of multiple persistent agents within a single interface. It provides a unified environment for chat, terminal access, and memory browsing while maintaining zero-fork compatibility with vanilla Hermes installations.

xaspx / hermes-control-interface

Acting as a multi-agent gateway, this self-hosted dashboard allows users to start, stop, and configure individual agent profiles from a central hub. It features an integrated xterm.js terminal with full PTY support and granular system metrics for monitoring the agent stack.

dodo-reach / hermes-desktop

A native macOS workspace that utilizes direct SSH connections to treat remote hosts as the single source of truth for agent data. The interface organizes the Hermes experience into specialized views for Kanban boards, cron jobs, and profile-aware session handling.

clawvader-tech / hermes-telegram-miniapp

This React-based mobile dashboard brings the Hermes experience into the Telegram ecosystem as a Mini App with a 10-page UI. It supports background spawning for up to five concurrent agent instances and includes real-time system monitoring gauges for CPU and memory usage.

aivrar / portable-hermes-agent

Designed for Windows users, this zero-install portable GUI requires no administrative rights and comes pre-loaded with over 100 tools for TTS, music, and ComfyUI. It features a unique dynamic tool maker that allows users to generate custom API wrappers at runtime.

fathah / hermes-desktop

This desktop companion simplifies the onboarding process with a guided installer for local dependency resolution and environment setup. It provides a centralized GUI for managing 16 messaging gateways and 14 distinct toolsets with integrated full-text search for session history.

sanchomuzax / hermes-webui

A monitoring-focused dashboard that provides a real-time activity feed for tracking agent sessions and associated costs. It specializes in transparency, offering FTS5 search capabilities for conversation history and read-only views for configuration and scheduled cron jobs.

Euraika-Labs / pan-ui

This chat-first workspace emphasizes extensibility through an integrated marketplace for discovering and installing new skills and MCP servers. It provides profile-scoped management of global memory and visualizes tool execution via SSE streaming timelines.

pyrate-llama / hermes-ui

Featuring a glassmorphic design, this React 18 application includes a Kanban-style task board to track the progress of active tools and subagents. It combines a lightweight Python proxy with an integrated file browser and terminal for workspace-aware project management.
