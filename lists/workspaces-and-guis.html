breakdown

farion1231 / cc-switch

A cross-platform desktop manager built with Tauri 2 that provides a unified interface for Hermes Agent alongside other major CLI agents like Claude Code and Gemini. It simplifies the lifecycle of multiple AI tools by offering a single, high-performance application for Windows, macOS, and Linux.

iOfficeAI / AionUi

This local open-source 'cowork' app provides a unified GUI for over 13 CLI agents, including Hermes Agent, with built-in web search and MCP tool integration. It stands out for its ability to automatically generate and edit professional documents like Morph PPTs, Word files, and Excel sheets.

nesquena / hermes-webui

A lightweight, zero-build web interface that brings full Hermes CLI parity to browsers and mobile devices using only Python and vanilla JavaScript. It features a functional three-panel layout that includes workspace file browsing and inline previews for a seamless remote experience.

fathah / hermes-desktop

A comprehensive desktop companion designed to guide users through the installation and configuration of local or remote Hermes backends. It offers visual management for personas, memory, and skills, while supporting an expansive 16 messaging gateways for broad connectivity.

EKKOLearnAI / hermes-web-ui

This dashboard focuses on multi-platform deployment, allowing users to configure and manage Hermes Agent across Telegram, Discord, Slack, WhatsApp, and Matrix. It includes robust usage analytics for tracking token consumption and costs alongside real-time session management.

outsourc-e / hermes-workspace

A native web workspace that introduces 'Swarm Mode' for orchestrating multiple persistent, role-based autonomous agents within a single interface. It provides a unified environment for chat, terminal access, and memory editing without requiring forks of the vanilla Hermes Agent.

dodo-reach / hermes-desktop

A specialized Mac workspace that connects to Hermes hosts via SSH to provide a native terminal and session experience. It includes productivity-focused views like Kanban boards and file explorers to manage agentic workflows directly from macOS.

xaspx / hermes-control-interface

A self-hosted operational control plane that features a three-panel Kanban pipeline for real-time swarm monitoring. It is particularly useful for power users needing deep oversight of MCP servers, system metrics, and detailed cost projections.

jazzyalex / agent-sessions

A privacy-focused macOS session browser that indexes local AI agent histories without telemetry or remote logging. Its 'Agent Cockpit' allows for live monitoring of active iTerm2 sessions, making it an ideal tool for auditing and searching past agent interactions.

nickvasilescu / hermes-desktop-os1

Designed for Orgo cloud computers, this macOS workspace offers one-click automated installation of Hermes Agent on fresh cloud VMs. It uniquely integrates WebRTC voice mode, enabling real-time vocal interaction with the agent via Orgo MCP.

liftaris / herm

An operator-focused TUI and dashboard built with OpenTUI that emphasizes efficiency through profile switching and session management. It features a native Eikon Marketplace, allowing users to easily search for and install shared agent extensions.

clawvader-tech / hermes-telegram-miniapp

A mobile-first React dashboard designed to run as a Telegram Mini App, offering secure HMAC-SHA256 authentication. It supports advanced mobile features like file attachments with automated vision analysis and OCR capabilities for on-the-go agent management.

Eynzof / hermes-agent-cn-desktop

A Windows-optimized desktop application built with Tauri and Rust that provides a localized interface for the Chinese Hermes community. It streamlines the management of the Hermes kernel, including automated updates and integrated UI for MCP tools and memory.

aivrar / portable-hermes-agent

A zero-install, 'no-admin' Windows GUI that bundles over 100 tools including TTS, MusicGen, and ComfyUI. It is designed for portability and ease of use, featuring a dynamic tool maker for creating custom API wrappers and Python handlers without Docker.

pyrate-llama / hermes-ui

A glassmorphic, single-file React application that uses a lightweight Python proxy to provide a modern web interface. It excels at real-time log streaming and live workspace browsing, offering a sleek way to manage skills and cron jobs.

sanchomuzax / hermes-webui

This dashboard serves as a monitoring and configuration hub, providing a real-time activity feed with cost tracking and model distribution data. It allows for read-only inspection of agent skill sources and full-text search across all conversation histories.

Euraika-Labs / pan-ui

A self-hosted workspace that prioritizes security and organization through isolated profile management for separate API keys and memory. It features a chat-first design with real-time SSE streaming and interactive tool approval cards for human-in-the-loop workflows.

Felix-Forever / hermes-agent-desktop

A multi-agent desktop client that orchestrates collaboration between 20 specialized AI agents. It includes a visual 'Skill Store' with over 50 curated one-click installations, making it easy to expand the agent's capabilities through a native interface.
