breakdown

mukul975 / Anthropic-Cybersecurity-Skills

This repository provides 754 structured cybersecurity skills mapped to industry-standard frameworks like MITRE ATT&CK and NIST. It enables Hermes to perform specialized tasks in cloud security and forensics while remaining compatible with the agentskills.io standard.

AMAP-ML / SkillClaw

SkillClaw introduces an 'Agentic Evolver' that automates skill deduplication and quality improvement through background evolution loops. It is designed to unify skill libraries across multiple devices and sessions, offering native integration for Hermes and Codex.

conorbronsdon / avoid-ai-writing

This tool audits and rewrites content to eliminate 36 distinct AI writing patterns and structural tells. It uses a comprehensive word replacement table and a two-pass audit process to ensure Hermes produces more natural, human-like prose.

wondelai / skills

A cross-platform library that offers modular installation of skills for architecture, UX design, and product strategy via a dedicated CLI. It provides Hermes with a standardized framework for professional services compatible with the agentskills.io ecosystem.

DougTrajano / pydantic-ai-skills

This project implements type-safe support for the Agent Skills open format specifically for Pydantic AI. It optimizes Hermes' performance by using progressive disclosure of skill metadata to significantly reduce token consumption.

Agents365-ai / drawio-skill

This skill allows Hermes to generate complex draw.io diagrams from natural language descriptions using specialized presets for machine learning and architecture. It features automated self-checking loops and supports advanced elements like animated connectors and tensor shapes.

Cranot / super-hermes

Super-Hermes empowers the agent to write its own analytical prompts and custom thinking instructions for complex problem-solving. It helps Hermes identify structural conservation laws in codebases and generate reports on its own analysis limitations.

smartcontractkit / chainlink-agent-skills

This library provides standardized skills for interacting with Chainlink oracles, including CCIP and Data Feeds. It includes a local A/B testing framework to help developers evaluate and refine how Hermes handles smart contract data.

tlehman / litprog-skill

This skill enables literate programming by weaving together prose, diagrams, and source code into synchronized markdown files. It allows Hermes to extract runnable code from documentation and automatically sync edits back to the primary document.

esaradev / icarus-plugin

Icarus facilitates cross-instance shared memory, allowing Hermes to remember past work and train replacement models. It features an end-to-end pipeline for extracting decision-quality data and fine-tuning models based on agent activity.

chigwell / skilldock.io

Skilldock is a registry and SDK for managing the lifecycle of reusable AI skills with recursive dependency resolution. It uniquely integrates commerce rails, allowing for skill transactions using both USD and the TON blockchain.

black-forest-labs / skills

These official skills provide Hermes with standardized prompting guidelines and API integration patterns for the FLUX image generation models. They support precise technical requirements like hex color specification and structured JSON prompting.

Romanescu11 / hermes-skill-factory

This meta-skill plugin monitors active workflows to automatically detect and propose new, reusable skills. It generates both documentation and Python scaffolds on the fly, allowing Hermes to expand its own capabilities during use.

ReinaMacCredy / maestro

Maestro is a harness for long-running agents that coordinates multi-step missions using milestones and validation gates. It persists agent state in a disciplined on-disk model and generates handoff briefs to bridge separate sessions.

tiann / execplan-skill

This skill focuses on the autonomous execution of complex, multi-step tasks with built-in checkpoints and recovery procedures. It provides Hermes with real-time progress tracking and rollback capabilities to ensure safe task completion.

cablate / Agentic-MCP-Skill

This progressive MCP client uses a three-layer lazy loading architecture to minimize token usage during tool discovery. Its socket-based daemon supports hot-reloading, maintaining persistent connections between Hermes and various MCP servers.

armelhbobdad / bmad-module-skill-forge

Skill Forge converts existing code repositories and developer discourse into version-pinned, agentskills.io-compliant skills. It ensures high reliability by providing verifiable audit trails with file-and-line level provenance for every generated skill.

amanning3390 / hermeshub

HermesHub serves as a community marketplace for browsing and installing skills with integrated security scanning and threat detection. It features an agent-to-agent feedback system to establish trust and reliability scores for shared skills.

PederHP / skillsdotnet

This project provides a C# and .NET implementation of the agentskills.io specification with full MCP integration. It allows Hermes to leverage .NET resources through progressive disclosure and standardized YAML frontmatter.

adnw-vinc / hermes-nextcloud

This skill grants Hermes the ability to manage files, calendars, and contacts within a Nextcloud environment via WebDAV and CalDAV. It ensures secure communication through HTTPS and includes a guided setup script for credential validation.
