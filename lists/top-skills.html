breakdown

mukul975 / Anthropic-Cybersecurity-Skills

This project provides 754 structured cybersecurity skills mapped to major industry frameworks like MITRE ATT&CK and NIST. It uses the standardized agentskills.io format to enable sub-second discovery and execution across 26 specialized security domains.

Agents365-ai / drawio-skill

This skill enables agents to generate complex draw.io diagrams from natural language descriptions, supporting formats like XML, PNG, and SVG. It features a self-check feedback loop to automatically fix layout issues across various diagram types including UML and ERDs.

conorbronsdon / avoid-ai-writing

This auditing tool uses a 109-entry replacement table and a two-pass audit system to eliminate detectable AI writing patterns. It identifies 42 distinct markers, such as significance inflation and copula avoidance, to ensure content reads naturally.

AMAP-ML / SkillClaw

SkillClaw automates the deduplication and quality improvement of agent skills across different devices and frameworks. It enables collective evolution by allowing user groups to share and refine learned skills within a unified library.

wondelai / skills

This is a cross-platform library compatible with Claude Code and agentskills.io-compliant agents. It offers modular installation of specialized frameworks for architecture, UX design, marketing, and product strategy via a dedicated CLI.

Romanescu11 / hermes-skill-factory

This meta-skill plugin monitors active workflows to automatically detect and propose new, reusable skills. It generates both Python scaffolds and documentation, allowing users to save and manage detected patterns via slash commands.

DougTrajano / pydantic-ai-skills

This implementation of the Agent Skills open specification is designed specifically for Pydantic AI. It utilizes progressive disclosure of metadata to reduce token usage while supporting both programmatic and filesystem-based skill definitions.

Cranot / super-hermes

Super-hermes teaches agents to write their own analytical prompts and custom thinking instructions for complex problem-solving. It helps identify structural trade-offs in codebases and generates reports to highlight analysis blind spots.

ReinaMacCredy / maestro

Maestro is a harness for long-running agents that utilizes a local-first Git-based state management system. It follows a structured 'spec-task-verify-ship' lifecycle and uses passive handoff envelopes for seamless context transfer between agents.

amanning3390 / hermeshub

HermesHub is a community marketplace for browsing and installing skills, featuring automated security scanning with over 65 threat detection rules. It includes an agent-to-agent feedback system and supports both crypto and Stripe-based payments for creators.

tlehman / litprog-skill

This literate programming skill allows agents to weave source code, prose, and diagrams into unified .lit.md files. It includes a tangler to extract runnable files and automatically syncs edits back to the primary documentation.

esaradev / icarus-plugin

Icarus facilitates cross-instance shared memory and the creation of replacement models through markdown-based directories. It features an end-to-end pipeline for extracting decision-quality training data to fine-tune and evaluate agent successors.

smartcontractkit / chainlink-agent-skills

This toolkit provides standardized skills for interacting with Chainlink oracle networks, including CCIP and Data Feeds. It includes CLI tools for local simulations and monitoring cross-chain smart contract workflows.

armelhbobdad / bmad-module-skill-forge

Skill Forge converts existing repositories and developer discourse into version-pinned, agentskills.io-compliant instructions. It provides verifiable audit trails with line-level citations to ensure the accuracy of generated agent skills.

black-forest-labs / skills

These are the official image generation skills for the FLUX model family, providing standardized prompting guidelines and API integration patterns. They support precise JSON-structured prompting and hex color specifications for high-fidelity visual outputs.

chigwell / skilldock.io

Skilldock.io is a registry for reusable AI skills that features recursive dependency resolution and an OpenAPI-driven SDK. It integrates a commerce flow supporting Stripe and TON blockchain for the management and discovery of agent capabilities.

tiann / execplan-skill

This skill enables autonomous execution of multi-step tasks with built-in checkpoints and recovery procedures. It tracks progress in real-time and provides rollback capabilities to ensure safe execution of complex, long-running operations.

cablate / Agentic-MCP-Skill

This progressive MCP client uses a three-layer lazy loading system to minimize token consumption when handling large toolsets. It maintains persistent connections to multiple MCP servers via a socket-based daemon for efficient tool inspection and retrieval.

PederHP / skillsdotnet

This C# and .NET implementation of the agentskills.io specification integrates with Microsoft.Extensions.AI for tool-based loading. It supports progressive disclosure via SKILL.md files and provides automatic registration for common skill directories.

adnw-vinc / hermes-nextcloud

This skill allows Hermes to manage Nextcloud files, calendars, and contacts using standard WebDAV and CalDAV protocols. It ensures secure communication via App Passwords and includes a guided setup script for easy configuration.
