breakdown

junhoyeo / tokscale

A Rust-based CLI tool that monitors token consumption across over 20 AI coding agents and CLIs. It provides real-time cost tracking via LiteLLM data and features a unique 3D visualization dashboard for usage analysis.

joeynyc / hermes-skins

This project offers community-driven visual customizations for the Hermes terminal interface. Users can define custom ASCII banners, spinners, and color schemes using YAML-based configurations that allow for partial theme overrides.

AlexAI-MCP / hermes-CCC

A specialized port that integrates the Hermes Agent brain and 46 core skills directly into the Claude Code Channel. It runs natively without OAuth or external processes and includes integrated MLOps tools for inference and RL training.

unmodeled-tyler / vessel-browser

An AI-native browser designed for autonomous agent control through the Model Context Protocol (MCP). It enables durable session management with checkpoints and provides a supervisory sidebar for human auditing and intervention.

Abruptive / Ankh.md

A framework for building Task-Agnostic Workflow (TAW) agents that operate within specific project scopes. It allows for per-folder configurations, enabling seamless switching between default and scoped agents with local memories and tools.

0xNyk / openclaw-to-hermes

A dedicated migration utility that facilitates the transition from OpenClaw to Hermes Agent. It automates the mapping of model configurations, personas, and memories, while also handling the transfer of local and system-level credentials.

roli-lpci / lintlang

A zero-latency static linter designed to identify structural bugs in AI agent configurations and system prompts. It uses seven specialized detectors to provide automated quality gating for CI/CD pipelines without requiring LLM calls or API keys.

42-evey / evey-setup

An automated deployment tool that spins up a complete Hermes Agent stack using Docker, LiteLLM, Ollama, and Qdrant. It provides a zero-cost environment featuring 29 community plugins and pre-configured routing for free local models.
