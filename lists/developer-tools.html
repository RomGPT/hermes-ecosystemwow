breakdown

junhoyeo / tokscale

This Rust-based TUI tracks token consumption and costs across multiple AI coding agents, including Hermes and Claude Code. It provides developers with local data visualization to monitor usage patterns and expenses across various model providers.

joeynyc / hermes-skins

This customization tool allows users to personalize the Hermes terminal interface through YAML-based themes, ASCII art, and custom spinner verbs. It supports partial skins that inherit from default themes and includes a variety of presets ranging from retro DOS to sci-fi aesthetics.

AlexAI-MCP / hermes-CCC

This project ports the Hermes Agent brain and 46 core skills directly into the Claude Code Channel for native execution without external processes or OAuth. It specifically enhances the environment with MLOps tools for vLLM, GGUF inference, and reinforcement learning training.

unmodeled-tyler / vessel-browser

Vessel is an AI-native browser designed for autonomous agent control, offering a human-visible UI for auditing agent actions in real-time. It exposes structured page schemas via MCP and maintains persistent sessions with annotated checkpoints for supervised browsing tasks.

Abruptive / Ankh.md

Ankh.md is a framework for creating scoped Hermes Agents that operate with per-folder configurations and project-local memories. It enables seamless switching between global and local agents, allowing for specialized tools and instructions tailored to specific directories.

roli-lpci / lintlang

This static linter improves agent performance by detecting structural flaws in system prompts and tool descriptions using seven deterministic pattern detectors. It provides concrete fix suggestions based on HERM v1.1 scoring without requiring API keys or LLM overhead.

0xNyk / openclaw-to-hermes

This migration utility streamlines the transition from OpenClaw to Hermes Agent by mapping personas, memories, and complex model configurations to the Hermes YAML format. It automates the transfer of workspace files and credentials from local environment files to ensure a smooth setup.

42-evey / evey-setup

This deployment script uses Docker to launch a complete Hermes Agent stack featuring LiteLLM, Ollama, and Qdrant for local GPU inference. It comes pre-configured with 29 community plugins for memory and social integration, focused on achieving a zero-cost setup using free models.
