breakdown

garrytan / gbrain

This project functions as an opinionated brain for Hermes Agent, utilizing a self-wiring knowledge graph with automated entity and relationship extraction. It offers a sophisticated hybrid search mechanism and includes 26 modular skills to manage data ingestion and enrichment.

vectorize-io / hindsight

Hindsight implements biomimetic data structures to facilitate long-term learning through dedicated retain, recall, and reflect workflows. It is designed for seamless integration with major LLM providers including OpenAI, Anthropic, and Ollama.

greyhaven-ai / autocontext

This recursive context harness accumulates validated knowledge and artifacts across repeated agent runs to improve performance on complex tasks. It utilizes a multi-agent loop to evaluate strategies and gate which knowledge is persisted for future missions.

stephenschoettler / hermes-lcm

A lossless context management plugin that uses a hierarchical DAG-based engine to ensure no message data is ever deleted. It features an immutable-first SQLite storage system and provides specialized tools for expanding and searching compacted history.

elkimek / honcho-self-hosted

This backend enables private, cross-session memory persistence through a self-hosted architecture. It supports any OpenAI-compatible provider or local inference server and includes automated setup scripts for Docker, PostgreSQL, and Redis.

yoloshii / ClawMem

A local-first, on-device memory engine that uses a unified SQLite vault to maintain persistence without cloud dependencies. It employs hybrid retrieval via vector search, BM25, and graph traversal to automate decision capture and handoff generation.

plur-ai / plur

Plur provides a shared memory layer for multi-agent systems using an open-format YAML engram structure. The system features zero API costs and implements automatic memory decay and strengthening based on how frequently information is used.

amanning3390 / flowstate-qmd

This anticipatory memory system uses an MCP-native server to prefetch context by monitoring agent session logs. It is a local-first solution utilizing SQLite-vec and GGUF models to provide proactive RAG capabilities.
