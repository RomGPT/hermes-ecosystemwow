breakdown

garrytan / gbrain

This opinionated brain builds a self-wiring knowledge graph by automatically extracting entities and relationships from agent interactions. It utilizes a hybrid search mechanism that combines vector retrieval with backlink-boosted ranking, supported by 26 modular skills for data ingestion and enrichment.

vectorize-io / hindsight

Hindsight implements biomimetic data structures to facilitate long-term agent learning through dedicated retain, recall, and reflect workflows. It allows Hermes to organize facts into evolving mental models while maintaining compatibility with major providers like OpenAI, Anthropic, and Ollama.

greyhaven-ai / autocontext

This recursive context harness uses a multi-agent loop to propose strategies and curate knowledge based on past performance. It specializes in distilling validated execution data into optimized runtimes, allowing agents to improve their behavior across repeated complex scenarios.

stephenschoettler / hermes-lcm

This plugin provides a lossless context engine that stores messages in an immutable SQLite DAG to ensure no data is ever deleted. It manages long histories through a three-level hierarchical summary system, offering specialized tools for agents to grep and expand compacted context.

elkimek / honcho-self-hosted

This self-hosted backend enables private, cross-session memory for Hermes Agent using a stack of Docker, PostgreSQL, and Redis. It is designed for users who require full control over their data while using any OpenAI-compatible or local inference server.

yoloshii / ClawMem

ClawMem is a local-first memory vault that uses a hybrid retrieval approach combining vector search, BM25, and graph traversal. It focuses on on-device persistence, automatically capturing agent decisions and generating handoffs without relying on cloud dependencies.

plur-ai / plur

Plur introduces a shared memory layer that stores persistent data as open-format YAML engrams to facilitate multi-agent interoperability. It features a zero-API-cost retrieval system that automatically decays or strengthens memories based on how frequently the agent accesses them.

amanning3390 / flowstate-qmd

This MCP-native server provides anticipatory memory by prefetching relevant context from session logs before the agent explicitly requests it. Built with SQLite-vec and node-llama-cpp, it offers a local-first RAG solution optimized for coding agent workflows.
