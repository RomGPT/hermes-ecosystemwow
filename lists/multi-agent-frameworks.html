breakdown

builderz-labs / mission-control

A self-hosted orchestration platform that manages multi-agent workflows across frameworks like CrewAI and AutoGen using a zero-dependency SQLite backend. It integrates the Aegis review system to provide human-in-the-loop oversight and spend monitoring for complex task dispatching.

swarmclawai / swarmclaw

This framework enables the creation of autonomous agent swarms with visual organizational charts and durable memory across 23+ providers, including Hermes and Ollama. It distinguishes itself by allowing agents to learn new skills directly from conversation history while managing delegation within the swarm.

howdymary / hermes-agent-metaharness

A specialized meta-optimization tool that uses an automated outer-loop search to evolve and improve harness code for Hermes Agent. It orchestrates TBLite and TB2 benchmark evaluations, tracking performance frontiers through structured candidate mutation and comparison.

runtimenoteslabs / gladiator

An agent-vs-agent arena where two competing AI companies, powered by Hermes Agent, battle for GitHub stars. The platform features a real-time dashboard to visualize task execution and scoreboard metrics as nine autonomous agents execute code and learning cycles.

1ilkhamov / opencode-hermes-multiagent

A comprehensive pipeline that coordinates 17 specialized agents through a master orchestrator to handle research, implementation, and infrastructure tasks. It enforces strict quality control by requiring mandatory sign-offs from dedicated reviewer and tester agents at every stage.

supermodeltools / bigiron

An AI-native software development lifecycle (SDLC) framework that provides Hermes with structural code graphs via the Supermodel MCP. It automates an eight-phase workflow, using graph-diff gates to ensure architectural integrity from initial planning to final health checks.

Rainhoole / hermes-agent-acp-skill

A delegation-focused framework that routes tasks between Hermes, Codex, and Claude Code while preventing context pollution through isolated workflows. It prioritizes operational safety by enforcing configurable timeouts and strict output character limits during agent handoffs.
