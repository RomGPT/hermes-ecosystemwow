breakdown

builderz-labs / mission-control

A self-hosted orchestration layer that uses SQLite to manage multi-agent workflows across frameworks like CrewAI and LangGraph without external dependencies. It provides centralized task dispatching, spend monitoring, and a human-in-the-loop Aegis review system for task sign-off.

swarmclawai / swarmclaw

This framework enables the creation of autonomous agent swarms featuring durable memory, scheduled tasks, and support for over 23 LLM providers. It distinguishes itself through visual orchestration, allowing users to manage delegation workflows via organizational charts.

howdymary / hermes-agent-metaharness

A specialized meta-optimization tool designed to improve harness code through an automated outer-loop search process. It orchestrates TBLite and TB2 benchmark evaluations to track performance frontiers via structured candidate mutation.

1ilkhamov / opencode-hermes-multiagent

A high-density coordination pipeline that manages 17 specialized agents across research, implementation, and infrastructure domains. It ensures reliability by enforcing mandatory quality gates through dedicated reviewer and tester agents.

runtimenoteslabs / gladiator

An agent-vs-agent arena where autonomous AI companies compete for GitHub stars using Paperclip and Hermes Agent. The platform features a real-time dashboard to visualize agent tasks, code execution, and competitive scoreboard metrics.

Rainhoole / hermes-agent-acp-skill

A delegation-focused framework that routes tasks between Hermes subagents, Codex, and Claude Code using an ACP-style approach. It maintains clean parent contexts through isolated workflows and enforces safety via configurable timeouts and character limits.

supermodeltools / bigiron

An AI-native software development lifecycle (SDLC) that executes an autonomous eight-phase process from planning to health checks. It leverages the Supermodel MCP to provide structural code graphs, using graph-diff phase gates to enforce architectural quality.
